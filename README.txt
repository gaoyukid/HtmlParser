HOW TO BUILD
1. make sure you have installed maven
2. run "build.sh"

HOW TO START SERVER
1. make sure you have properly built the project. to verify, check if there is any content in "target" folder

HOW TO USE
1. start your jetty service via SampleJettyApplication
2. http://localhost:8080/html?url=%YOUR_URL_FULLPATH%

VARIATIONS

localhost:8080/html?url=http://www.nba.com&format=title
localhost:8080/html?url=http://www.nba.com&format=text
localhost:8080/html?url=http://www.nba.com&format=json
